# 📦 IMS Pro - Frontend Application

![CI Status](https://github.com/JackScoble/ims-frontend/actions/workflows/frontend-ci.yml/badge.svg)

Welcome to the frontend repository for **IMS Pro**, a robust Inventory Management System. This React Single Page Application (SPA) provides a modern, responsive, and intuitive interface for managing stock, processing orders, analyzing trends, and tracking system audits.

✅ **Live Frontend:** [Render Deployed Fronted](https://ims-frontend-9xju.onrender.com/login)

🖥️ **Backend Repository:** [JackScoble/ims-backend](https://github.com/JackScoble/ims-backend)

---

## 🛠️ Tech Stack

* **Framework:** React (Bootstrapped with Vite for rapid development)
* **Routing:** React Router v6
* **HTTP Client:** Axios (Configured with automated JWT refresh interceptors)
* **Testing:** Vitest & React Testing Library
* **Notifications:** React Hot Toast

---

## ✨ Key Features

* **Secure Authentication Flow:** Complete login, registration, and password reset journeys with silent JWT token refreshing to keep users securely logged in without interruption.
* **Interactive Dashboard:** Real-time inventory filtering, searching by Name/SKU, and quick-toggle views for low-stock items. Features seamless CRUD modals for rapid data entry.
* **Failsafe Order Processing:** A dedicated transaction interface that validates order quantities against live server stock to prevent overselling before the request even leaves the client.
* **Rich User Profiles:** Customizable user settings including role displays, secure avatar uploads (with instant blob previews), and global Light/Dark theme preferences powered by React Context.
* **Audit Trail Visibility:** Direct access to the backend's audit engine, allowing users to view the granular history (who, what, and when) of specific inventory items directly from the dashboard.
* **Analytics & Visualizations:** Interactive charting to digest the time-series financial snapshots generated by the API.

## 🚀 Local Setup Instructions

Follow these steps to get the React application running on your local machine.

### 1. Clone the Repository

```bash
git clone https://github.com/JackScoble/ims-frontend.git
cd ims-frontend
```

### 2. Install Dependencies

Make sure you have Node.js installed, then run:

```bash
npm install
# or if you use yarn:
# yarn install
```


### 3. Start the Development Server

Launch the Vite development server:

```bash
npm run dev
```

The application will now be available, typically at `http://localhost:5173/`.
